# Janvi Patel - Professional Portfolio Website

An advanced, premium, and recruiter-focused portfolio website for **Janvi Patel**, tailored for roles in **Data Engineering**, **Data Analytics**, and **Software Engineering**. Built using **HTML, Tailwind CSS, and JavaScript**, this portfolio is fully responsive, interactive, and optimized for career discovery.

---

## 🚀 Live Demo & Visuals

The portfolio features:
- **Premium Dark Aesthetics**: Styled with deep graphite and slate backgrounds, glassmorphism panels, and neon-technical accent glows.
- **Micro-Animations**: Real-time typing effects, IntersectionObserver-triggered scroll reveals, and glow transformations on hovered elements.
- **Recruiter Utilities**: Sticky navigation, mobile-responsive layouts, smooth scrolling, filtering project lists, and a live-validated contact form.

---

## 🛠️ Technology Stack

- **Core**: HTML5, Vanilla JavaScript (ES6+)
- **Styling**: Tailwind CSS (Tailwind CLI build system)
- **Icons**: Lucide Icons (SVG CDN integration)
- **Fonts**: Google Fonts (Outfit & Inter)

## ⚙️ Setup & Local Development

To run this project locally, ensure you have **Node.js** installed on your system.

### 1. Install Dependencies
Initialize the packages and install the Tailwind CSS compiler:
```bash
npm install
```

### 2. Compile CSS Stylesheets
You can build the production stylesheet or watch for changes during development.

- **To run a single compile build**:
  ```bash
  npm run build
  ```

- **To watch changes in real-time (hot development)**:
  ```bash
  npm run watch
  ```

### 3. Run the Application
Open `index.html` directly in your web browser, or use a local dev server (e.g., VS Code Live Server, Python HTTP server, or Node `http-server`) to preview the site.
```bash
# Example using Python's built-in HTTP server
python -m http-server 8000
```
Then navigate to `http://localhost:8000`.

## 🎨 How to Customize Content

### 📑 1. Update the Resume
- Replace the placeholder PDF at `assets/Janvi_Patel_Resume.pdf` with your actual resume file. Ensure the filename matches exactly, or update the references in `index.html`:
  ```html
  <a href="./assets/Janvi_Patel_Resume.pdf" download>...</a>
  ```

### 🖼️ 2. Update the Profile Photo
- Replace `assets/profile.jpg` with your professional headshot.

### 💻 3. Customize Project Showcases
Project cards are stored in `index.html` inside the `#projects` section.
- **To change tech stack badges**, modify the span tags inside the card:
  ```html
  <span class="px-2.5 py-0.5 rounded text-[10px] font-mono bg-slate-900 border border-white/5 text-cyan-400">YourTech</span>
  ```
- **To configure project filters**, update the `data-category` attribute on the corresponding `.project-card` div:
  ```html
  <div class="project-card ... " data-category="data-engineering cloud">
  ```
  *Categories include: `data-engineering`, `analytics`, `cloud`, `software`.*

### 📬 4. Form Action Setup
The contact form is controlled by `src/script.js`.
- For production use, you can integrate it with a service like [Formspree](https://formspree.io/) or [Web3Forms](https://web3forms.com/). Update the `<form>` opening tag in `index.html`:
  ```html
  <form action="https://formspree.io/f/YOUR_FORM_ID" method="POST" id="contact-form" ...>
  ```
